# JPA Query Project - Submission 1

This is a small Java project that uses JPA with Hibernate and Derby.

The program creates authors, books, and readers, and then runs different JPA queries:
- get books by author
- get readers of a book
- get authors whose books have been read
- count books per author
- get books by genre using a Named Query

Run the "TestConnection" class to see the results.
